# FitMate Design System - Premium Fitness UI

## Color Palette

### Primary Colors
- **Primary Orange**: `#FF6B35` (RGB: 255 107 53) - Main CTA, accents, energy
- **Secondary Teal/Cyan**: `#00D4FF` (RGB: 0 212 255) - Highlights, secondary accents
- **Gold Accent**: `#FFD60A` (RGB: 255 214 10) - Warnings, intensity medium, tertiary accents

### Neutral Colors
- **Background**: Pure Black `#000000` - Main background
- **Card**: `#0D0D0D` (RGB: 13 13 13) - Elevated surfaces
- **Secondary**: `#191919` (RGB: 25 25 25) - Form inputs, alternative backgrounds
- **Border**: `#333333` (RGB: 51 51 51) - Subtle dividers
- **Muted**: `#666666` (RGB: 102 102 102) - Muted text
- **Foreground**: Pure White `#FFFFFF` - All text

## Typography

### Font Stack
- System fonts with fallbacks: `-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Helvetica Neue', 'Roboto', sans-serif`
- Optimized for readability and modern appearance

### Hierarchy
- **H1**: 36px, 700 weight, -0.5px letter-spacing
- **H2**: 30px, 700 weight, -0.5px letter-spacing
- **H3**: 24px, 700 weight, -0.5px letter-spacing
- **Body**: 14-16px, 400 weight, -0.3px letter-spacing
- **Labels**: 12px, 600 weight, uppercase tracking

## Component Styling

### Cards
- **Card Elevated**: Dark card with subtle border and orange shadow glow
- **Glass Effect**: Semi-transparent dark overlay with backdrop blur
- Border color: `#333333` with 80% opacity
- Shadow: `0 4px 12px rgba(0, 0, 0, 0.6), 0 1px 3px rgba(255, 107, 53, 0.1)`

### Buttons
- **Primary**: Orange background `#FF6B35` with white text
- **Hover**: Lighter orange `#FF8A5E`
- **Ghost**: Transparent with hover background

### Forms
- **Input Background**: `#191919` (RGB: 25 25 25)
- **Border**: `#333333` with interactive focus states
- **Focus Ring**: 2px ring with primary color at 30% opacity

### Progress Bars
- **Container**: `#191919` (RGB: 25 25 25)
- **Fill**: Gradient from orange to gold with shadow glow

## Animations

### Transitions
- **Smooth**: 300ms ease-in-out for all interactive elements
- **Fade In**: 500ms fade from below
- **Slide In**: 500ms slide from sides

### Hover Effects
- Border color brightens to primary
- Slight shadow enhancement
- Text color shifts toward primary

## Spacing & Radius

- **Border Radius**: 0.75rem (12px) for cards, 0.5rem for buttons
- **Padding**: 1.5rem (24px) for cards, 1rem (16px) for sections
- **Gap**: 1rem (16px) standard spacing

## Design Principles

1. **Energy & Performance**: Orange primary conveys fitness energy
2. **Modern Professional**: Cyan accents add sophistication
3. **High Contrast**: White on black ensures readability
4. **Hierarchical**: Clear visual hierarchy guides attention
5. **Spacious**: Generous spacing prevents clutter
6. **Interactive**: Hover states and smooth transitions enhance UX
